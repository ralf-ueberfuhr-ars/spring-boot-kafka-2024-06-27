## Bauen und Starten

Du kannst die jeweiligen Projekte bauen und starten mit folgenden Befehlen:

```bash
# Ins jeweilige Projekt wechseln
cd <ordner>
# Bauen
mvn package
# Starten
java -jar target/<file>.jar
# oder
mvn spring-boot:run
```
