# Account Service Provider

Diese Anwendung implementiert eine REST-API zur Verwaltung von Kundendaten.
([OpenAPI](src/main/resources/static/openapi.yml))

Starte die Anwendung, dann hast Du folgende URLs:
- http://localhost:8080/index.html (Startseite)
- http://localhost:8080/db (H2 Console)
- http://localhost:8080/api/v1/customers (Kundendaten/JSON)
